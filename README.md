# 智能会议助手

一个让会议更高效的智能工具，通过二维码收集会议内容，利用AI分析总结讨论重点。

## 🚀 功能特性

### 核心功能
- **二维码扫码输入**: 会议前展示二维码，参会者扫码即可输入会议内容
- **多方式输入支持**: 支持文字输入和语音输入两种方式
- **智能AI分析**: 自动分析会议内容，提取讨论重点和行动项
- **历史会议回顾**: 保存所有会议记录，随时查看历史会议内容
- **待办事项管理**: 跟踪会议确定的行动项，支持完成状态标记
- **实时统计展示**: 实时显示参会人数和内容提交数量

### 优质体验
- **响应式设计**: 完美适配桌面端和移动端
- **流畅动画**: 丰富的交互动画和微交互效果
- **模板快速输入**: 提供常用会议内容模板，提高输入效率
- **本地数据存储**: 所有数据保存在本地，保护隐私安全

## 📦 技术栈

- **前端框架**: HTML5, CSS3, JavaScript (ES6+)
- **样式框架**: Tailwind CSS v3
- **二维码生成**: QRCode.js
- **图标库**: Font Awesome 6
- **本地存储**: localStorage API
- **语音识别**: Web Speech API (模拟实现)

## 📱 使用方法

### 会议主持人操作流程

1. **启动会议**
   - 打开 `index.html` 文件
   - 页面会自动生成本次会议的二维码和会议编号

2. **展示二维码**
   - 将二维码展示给所有参会人员
   - 参会者使用手机扫码即可打开输入页面

3. **实时监控**
   - 页面实时显示已提交人数和内容数量
   - 可以点击"刷新二维码"生成新的会议二维码

4. **分析会议内容**
   - 当所有人提交完成后，点击"分析会议内容"
   - AI会自动分析并展示讨论重点和行动项

5. **保存会议记录**
   - 分析完成后，会议记录会自动保存到历史记录
   - 可以随时查看和回顾

### 参会人员操作流程

1. **扫码输入**
   - 使用手机相机扫描会议二维码
   - 自动跳转到会议内容输入页面

2. **填写信息**
   - 输入您的姓名
   - 输入会议汇报内容，可以选择：
     - 直接文字输入
     - 使用语音输入（点击麦克风按钮）
     - 使用快速模板

3. **提交内容**
   - 点击"提交内容"按钮
   - 提交成功后可以继续添加新内容

## 🛠️ 自定义配置

### 修改会议ID生成规则

在 `script.js` 中修改 `generateMeetingId()` 方法：

```javascript
generateMeetingId() {
    // 自定义您的会议ID生成规则
    const date = new Date();
    const dateStr = date.toISOString().slice(0, 10).replace(/-/g, '');
    const randomStr = Math.random().toString(36).substring(2, 8).toUpperCase();
    return `MEET-${dateStr}-${randomStr}`;
}
```

### 修改AI分析逻辑

在 `script.js` 中修改 `simulateAIanalysis()` 方法，自定义分析规则和结果格式。

### 自定义样式主题

在 `styles.css` 中修改颜色变量和样式类，打造符合您品牌的界面。

## 📊 数据管理

### 数据存储位置
- 所有会议数据保存在浏览器的 localStorage 中
- 会议内容: `meeting_{会议ID}_contents`
- 参会人员: `meeting_{会议ID}_participants`
- 历史会议: `meetingsHistory`
- 待办事项: `meetingTodos`

### 数据导出

可以通过浏览器开发者工具导出数据：

1. 打开浏览器控制台 (F12)
2. 输入以下命令：

```javascript
// 导出所有历史会议
console.log(JSON.parse(localStorage.getItem('meetingsHistory')));

// 导出待办事项
console.log(JSON.parse(localStorage.getItem('meetingTodos')));
```

- **本地存储**: 所有数据仅保存在用户本地浏览器中

- **自主控制**: 用户完全控制数据的保存和删除
